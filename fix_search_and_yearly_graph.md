## 修正1: 検索ボタンと検索機能の復帰

### 対象ファイル: `src/components/Bubble/BubbleSearch.tsx` / `BubbleSearch.css`（存在しなければ新規作成）
### 追加対象: `src/components/Sky/SkyView.tsx` / `SkyView.css`

検索機能が消えています。以下の仕様で復帰してください。

**検索ボタン:**
- 画面右下（入力バーの上あたり）に小さな検索アイコン（🔍）を常時表示
- `position: fixed` でスクロールに影響されない
- 泡が10個以下のときは非表示
- 他のボタン（「今日の空」「一覧」）と重ならない位置に配置

**検索パネル（検索ボタンをタップで開く）:**
- 画面下部からスライドアップするパネル
- パネル内に、現在空にある泡のテキスト一覧をリスト表示
- フリーテキスト入力欄はなし。一覧から選ぶだけ
- 並び順: 作成日が新しい順
- 「キープ中」の泡は項目の横に小さなマーク
- パネル外をタップで閉じる
- デザイン: すりガラス風の背景（`backdrop-filter: blur`）

**セレクト後の挙動:**
- 一覧から泡を選択する
- 検索パネルが閉じる
- 空が自動スクロールして、選択された泡が画面中央に来る
- 選択された泡が2秒間キラッと光る（`box-shadow` のglow を一時的に強くする）
- 2秒後に光は元に戻る

**重要:** 検索パネルから泡を選んだとき、BubbleDetail（タスク詳細のポップアップ）は開かない。空の画面上でシャボン玉が光るだけ。

## 修正3: 「今年できたこと」タブの追加

### 対象ファイル: `src/components/DailySky/DailySkyView.tsx` / `DailySkyView.css`

「今日の空」画面に、タブ切り替えで「今年できたこと」ビューを追加する。

**タブ構成:**

```
[ 今日の空 ]  [ 今年できたこと ]
```

- 画面上部にタブを2つ並べる
- デフォルトは「今日の空」（既存の画面がそのまま表示される）
- 「今年できたこと」をタップすると切り替わる
- タブのデザイン: 控えめなテキストタブ。選択中のタブに下線

**「今年できたこと」ビューの内容:**

GitHubのコントリビューショングラフに近いイメージ。ただし、もっとシンプルでやさしい見た目。

**レイアウト:**
```
      1月  2月  3月  4月  5月  6月  7月  8月  9月  10月 11月 12月
月
火
水
木
金
土
日
```

- 横軸: 週（1月1日の週から現在の週まで）
- 縦軸: 曜日（月〜日）
- 各セルは小さな四角（8px × 8px 程度、角丸2px）
- 月の区切りに月名ラベル（Jan, Feb... または 1月, 2月...）

**セルの色:**
- done ログが0の日: 薄いグレー（`#ebedf0` 程度）
- done ログが1の日: 薄い水色（`#9be9a8` ではなく、Pukariのテーマに合わせて `#b3e0f2` 程度）
- done ログが2〜3の日: やや濃い水色（`#7cc4e0`）
- done ログが4以上の日: 濃い水色（`#3a9bc5`）
- 未来の日: 非表示（セルを描画しない、または完全に透明）

**色のトーン:**
- GitHubの緑ではなく、Pukariの空のテーマに合わせた**水色〜青のグラデーション**
- 派手にしない。ぽつぽつと色がつく控えめな感じ
- 凡例は右下に小さく: `少ない ■■■■ 多い`

**データ取得:**
- `bubble_logs`（または localStorage の `pukari-logs`）から、今年の全 done ログを取得
- 日付ごとに done 数をカウント
- 年の切り替えは不要（今年だけ表示）

**下部に統計は表示しない。** グラフだけ。「今年◯日できた」のような数字は出さない（プレッシャーになるため）。ただし、グラフを見れば「ぽつぽつ色がついている」ことが自然に分かる。

## 新規作成: ContributionGraph コンポーネント

### 新規ファイル: `src/components/DailySky/ContributionGraph.tsx` / `ContributionGraph.css`

**Props:**
```typescript
interface ContributionGraphProps {
  year: number;
  doneCountByDate: Record<string, number>;  // { '2026-01-15': 3, '2026-03-22': 1, ... }
}
```

**描画ロジック:**
- 1月1日の曜日から開始して、週ごとに列を作る
- 各列は7行（月〜日）
- 今日以降の日は描画しない
- 全体を横スクロール可能にする（1年分は画面幅に収まらない場合がある）
- ただし、現在の週が画面に見えるよう、初期スクロール位置を右端（最新）に設定

**サイズ調整:**
- セルサイズ: 8px × 8px
- セル間ギャップ: 2px
- 全体幅: 約 53週 × 10px = 530px 程度
- スマホ幅（375px）では横スクロールになる
