## 修正5: PWA解説ページ

### 新規ファイル: `src/components/Pages/PwaGuidePage.tsx` / `PwaGuidePage.css`

**アクセス方法:** サイドメニュー →「ホーム画面に追加する」

**画面構成:**

全画面オーバーレイ。左上に「←戻る」ボタン。

**内容:**

---

ヘッダー:
```
ホーム画面に追加する
Pukariをアプリのように使えます
```

---

iPhone（Safari）の場合:

```
1. Safariで Pukari を開く
2. 下部の共有ボタン（□↑）をタップ
3. 「ホーム画面に追加」を選択
4. 「追加」をタップ

ホーム画面にPukariのアイコンが追加されます。
次回からアイコンをタップするだけで開けます。
```

---

Androidの場合:

```
1. Chromeで Pukari を開く
2. 右上の「︙」メニューをタップ
3. 「ホーム画面に追加」または「アプリをインストール」を選択
4. 「インストール」をタップ

ホーム画面にPukariのアイコンが追加されます。
```

---

PCの場合:

```
1. Chromeで Pukari を開く
2. アドレスバーの右端にあるインストールアイコン（↓）をクリック
3. 「インストール」をクリック
```

---

補足:
```
ホーム画面に追加すると、
ブラウザのアドレスバーが消えて
アプリのように全画面で使えます。
```

---

**デザイン:**
- 各ステップは番号付きで、1ステップずつ分かりやすく
- iPhone / Android / PC をタブまたはアコーディオンで切り替え
- スクリーンショットは今は入れない（テキストだけで十分伝わる）
- 背景: 白。読みやすさ優先

## 確認方法

1. 開発サーバーを起動する
2. ヘッダーが表示され、左に「Pukari」、右に「☰」があることを確認
3. 「☰」をタップ → サイドメニューが開くことを確認
4. 「Pukariについて」→ 概要ページが開き、開発背景が読めることを確認
5. 「←戻る」でメイン画面に戻れることを確認
6. 「ホーム画面に追加する」→ PWA解説ページが開くことを確認
7. iPhone / Android / PC の説明が読めることを確認
8. フッターに検索・ブルーラグーン・今日の空のボタンが並んでいることを確認
9. 検索ボタンが泡1個から表示されることを確認
10. ヘッダー・フッターが空の景色を邪魔していないことを確認
11. ブルーラグーン画面ではヘッダーが表示されないことを確認
12. 「確認完了しました。サーバーを停止します。」と報告してからサーバーを停止する
13. **私の指示を待つ**
