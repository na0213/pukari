## 修正1: 星空（night）にキラキラ星を追加

### 対象ファイル
- `src/hooks/useSkyColor.ts`
- `src/styles/animations.css`
- `src/components/Sky/SkyView.tsx`
- `src/components/Sky/SkyView.css`

### 内容

`useSkyColor.ts` で `SkyPhase` が `night`（できた数 7+）のとき、空の背景に小さな星がキラキラ光るエフェクトを追加してください。

**星の仕様:**
- 小さな白い点（2px〜4px）がランダムな位置に 15〜20個 表示される
- 各星がゆっくり明滅する（`opacity` を 0.3〜1.0 で繰り返す）
- 各星の明滅タイミングはランダムにずらす（`animation-delay` をランダム）
- `prefers-reduced-motion` の場合は明滅せず、静的に表示する

**実装方法:**
- `SkyView.tsx` で `skyPhase === 'night'` のときだけ、星のレイヤーを描画する
- 星は CSS の `::after` 疑似要素または別の div レイヤーで実装
- `animations.css` に `@keyframes twinkle` を追加:

```css
@keyframes twinkle {
  0%, 100% { opacity: 0.3; }
  50% { opacity: 1; }
}
```

- 星の位置はコンポーネントマウント時にランダム生成し、state で保持する
- シャボン玉の描画より背面（`z-index` を低く）に配置する

## 修正2: drifted（遠くへ流れた）泡が画面から消えてしまう問題

### 対象ファイル
- `src/components/Bubble/BubbleItem.tsx`
- `src/components/Bubble/BubbleItem.css`

### 問題
`drifted` 状態の泡が画面外に移動してしまい、完全に見えなくなっている。
ユーザーが「遠くへ流れた泡」を見つけられないと、復活させることもできない。

### 修正内容

`drifted` 状態の泡は、画面から消すのではなく **「空の端に小さく留まっている」** 状態にしてください。

**位置の調整:**
- `drifted` の泡は画面の下端付近（bottom: 10%〜15% あたり）に集まるように配置する
- 画面の左右にランダムに散らばる（完全に端にはしない。画面内に収める）
- 画面外にはみ出さないように `position` を制限する

**サイズの調整:**
- 通常の泡の **40%** 程度のサイズ（30px〜50px）にする
- 現在の設定が小さすぎるなら大きくする

**見た目の調整:**
- 半透明（opacity: 0.4〜0.5）にする
- ふわふわアニメーションは維持するが、動きの幅をさらに小さくする（振幅 3px 程度）
- タップ可能であることが分かるように、タップすると BubbleDetail が開く動作は維持する

**確認ポイント:**
- スマートフォン幅（375px）で `drifted` の泡が画面内に見えていること
- `drifted` の泡をタップして詳細を開き、「戻す」操作ができること
- `drifted` の泡が 5個以上あっても重なりすぎず、区別できること
