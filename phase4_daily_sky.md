# Phase 4:「今日の空」ジャーナル

## このフェーズの目的
「今日の空」画面を実装する。これがこのアプリの本丸。「何もできていない」と思っていたけど、実は触れていた・できていた、と気づくための場所。

## 前提

- CLAUDE.md の安全ルールに従うこと

## 指示

### 1. DailySkyView コンポーネント（`src/components/DailySky/DailySkyView.tsx`）

「今日の空」画面:

**アクセス方法:**
- メイン画面（空）の下部にある「今日の空」ボタンをタップ
- 画面が下からスライドアップする（フルスクリーンオーバーレイ）

**画面構成:**

#### 上部: 今日の空のビジュアル
- 画面上部1/3に、現在の空の色（SkyPhase）をグラデーションで表示
- その中央に、今日の「できた」数と「触れた」数を表示
- 例: 「できた 3 ・ 触れた 5」
- フォントは控えめなサイズ。強調しすぎない

#### 中部: できたこと一覧
- 今日「できた！」にしたシャボン玉のテキストを一覧表示
- 各項目の横に小さなシャボン玉アイコン（空に溶けた色）
- リストが空の場合: 「まだ何もなくても、大丈夫。」と表示

#### 下部: 触れたこと一覧
- 今日「触れた」シャボン玉のテキストを一覧表示
- 各項目の横に小さなシャボン玉アイコン（ほんのり光る色）
- セクション見出し: 「触れたこと」（控えめなフォント）
- リストが空でも何も表示しない（存在しないセクションとして扱う）

#### 最下部: 閉じるボタン
- 「空に戻る」ボタンで元の空画面に戻る

**トーンの注意:**
- 数字を大きく強調しない（プレッシャーになる）
- 「触れただけ」も表示することで「ゼロではない」を伝える
- 空が「早朝の薄い青」（何もしていない日）でも、否定的なメッセージは出さない
- 統計・ストリーク・グラフは一切表示しない

### 2. useDailySky フック（`src/hooks/useDailySky.ts`）

日ごとの空の記録を管理:

```typescript
interface UseDailySkyReturn {
  todaySky: DailySky;           // 今日の空の状態
  pastSkies: DailySky[];        // 過去の空（直近7日分、無料版）
  getSkyForDate: (date: string) => DailySky | null;
}
```

**データの計算方法:**
- 「今日」の判定は日本時間（JST）の0:00〜23:59
- `doneCount`: 今日 `doneAt` が記録された泡の数
- `touchedCount`: 今日 `touchedAt` が記録された泡の数（doneも含む）
- `phase`: `useSkyColor(doneCount)` で算出

**保存:**
- localStorage に `pukari-daily-skies` キーで保存
- 日付ごとに1レコード

### 3. 過去の空ビュー（簡易版）

「今日の空」画面を左にスワイプすると、昨日の空が見える。
さらにスワイプすると、一昨日...と7日分見られる。

**各日の表示:**
- その日の空の色（グラデーション背景）
- 日付
- できた数・触れた数
- 泡のテキスト一覧

**スワイプ実装:**
- framer-motion の `drag="x"` を使用
- スナップポイントで日ごとに切り替わる

### 4. 空に戻る導線

「今日の空」画面から空に戻るときのアニメーション:
- 下にスワイプ、または「空に戻る」ボタン
- 画面が下にスライドして消える
