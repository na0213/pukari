## 修正3:「できた！」ボタンの文言を状態で切り替え

### 対象ファイル: `src/components/Bubble/BubbleDetail.tsx` / BubbleDetail のCSS

**くりかえし OFF（repeatable = false）の場合:**
- ボタン文言: 「できた！」（今まで通り）
- 押すと泡が完了（空に溶ける）

**くりかえし ON（repeatable = true）かつ、今日まだできていない場合:**
- ボタン文言: 「今日もできた！」
- ボタンの見た目: 緑の背景、白テキスト、押せる状態
- 押すと bubble_logs に記録。泡は残る。ボタンが下記に変わる

**くりかえし ON（repeatable = true）かつ、今日すでにできた場合:**
- ボタン文言: 「✓ 今日はできた」
- ボタンの見た目: 薄いグレー背景、グレーテキスト、disabled（押せない）
- ✓マーク付き

## 確認方法

1. 開発サーバーを起動する
2. 泡をタップして詳細を開く → 「遠くへ流す」が表示されないことを確認
3. メイン画面で泡を長押し → 「遠くへ流す」メニューが出ることを確認
4. 「くりかえす」を ON にする → 「毎日「できた！」を記録できます」が表示されることを確認
5. 「くりかえす」を OFF にする → キャプションが消えることを確認
6. くりかえし ON の泡で「今日もできた！」を押す → ボタンが「✓ 今日はできた」に変わり、泡が残っていることを確認
7. くりかえし OFF の泡で「できた！」を押す → 泡が空に溶けることを確認
8. 「確認完了しました。サーバーを停止します。」と報告してからサーバーを停止する
